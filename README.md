# Reflective

## Development

```sh
# Install the latest version of `node` (probably through `nvm`)
nvm install latest
nvm use <latestVersion>

# Install the latest version of `pnpm`
npm install pnpm@latest --global

# Install dependencies
pnpm install

# Run
pnpm run start

# Build
pnpm run build
```
